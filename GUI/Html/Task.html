Постановка задачи

Списки в языке Kotlin являются структурой данных, предназначенной для хранения набора значений в определённой последовательности.

Для объявления списка с инициализацией в языке Kotlin используется ключевое слово val, имя переменной, оператор присваивания и вызов функции listOf.

Формат записи:

val имя_переменной = listOf(элемент1, элемент2, ...);

Примеры:

val animals = listOf("Dog", "Cat", "Cow");
val empty = listOf();

В связи с разработанной контекстно-свободной грамматикой G[Z] синтаксический анализатор объявления списка с инициализацией на языке Kotlin будет считать верными следующие специфические записи: 

val animals = listOf("Dog", "Cat", "Cow");
val nums = listOf(1, -2, +3.5, 0);
val flags = listOf(true, false, true);
val letters = listOf('a', 'b', 'c');
val empty = listOf();
